# Hama-API

## Setup
### Step 1:
```shell
$ pip install virtualenv
```
### Step 2:
```shell
$ python -m venv env
```
### Step 3:
```shell
$ env\Scripts\activate
```
### Step 4:
```shell
$ pip install -r requirements.txt
```
### Step 5:
```shell
$ python main.py
```
### Export pakages
```shell
$ pip freeze > requirements.txt
```
